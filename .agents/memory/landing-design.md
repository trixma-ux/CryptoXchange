name: Landing page design
description: Design direction for CryptoXchange landing page
---

The user explicitly requested a design inspired by Trust Wallet (screenshot: image_1781517485299.png).

**Design:** White background, bold black typography (#0f172a), purple/indigo gradient accents (linear-gradient(135deg, #6366f1, #8b5cf6)), clean minimal nav, ticker bar on dark (#0f172a), hero with text left + interactive card right.

**Why:** Previous dark amber (#f59e0b) theme was the original crypto dark theme. User rejected it in favor of cleaner white/modern design.

**How to apply:** All new pages (dashboard, auth, etc.) should lean toward white backgrounds and purple/indigo accents rather than dark amber. index.css still has the dark theme classes — prefer inline styles or new utility classes for redesigned pages.
